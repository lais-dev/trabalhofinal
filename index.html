Sistemas Numéricos

Vamos entender um pouco mais do que se trata os Sistemas Numéricos:

Os sistemas de numeração são métodos utilizados para representar números de forma sistemática. Os três sistemas de numeração mais comuns são o binário, o decimal e o hexadecimal. Aqui estão algumas informações sobre cada um deles:

1. Sistema Binário:

O sistema binário é um sistema de numeração baseado em 2. Ele utiliza apenas dois dígitos: 0 e 1. Cada dígito binário é chamado de bit (abreviação de binary digit). A sequência de bits pode ser usada para representar qualquer número ou informação digital. Os computadores digitais são baseados no sistema binário, pois podem armazenar e manipular informações na forma de bits. Os valores no sistema binário seguem uma progressão exponencial de base 2. Por exemplo:

  • 10101 em binário representa o número decimal 21.
  • 1101 em binário representa o número decimal 13.

2. Sistema Decimal:

O sistema decimal é o sistema de numeração mais utilizado no cotidiano. É um sistema de numeração baseado em 10, o que significa que ele utiliza 10 dígitos distintos: 0, 1, 2, 3, 4, 5, 6, 7, 8 e 9. Cada dígito tem um valor que depende da posição em que ele está localizado dentro do número. A posição mais à direita é a posição das unidades, seguida pelas dezenas, centenas, milhares e assim por diante. O valor de cada posição é uma potência de 10. Por exemplo:

  • 735 em decimal representa o número 7 centenas, 3 dezenas e 5 unidades, totalizando o número decimal 735.

3. Sistema Hexadecimal:

O sistema hexadecimal é um sistema de numeração baseado em 16. Ele utiliza 16 dígitos distintos: 0 a 9 e as letras A a F (onde A representa o valor 10, B representa 11 e assim por diante até F, que representa o valor 15). O sistema hexadecimal é frequentemente usado em programação e computação, pois é uma forma compacta de representar sequências de bits. Cada dígito hexadecimal representa quatro dígitos binários. Por exemplo:

  • 2A em hexadecimal representa o número binário 00101010, que por sua vez representa o número decimal 42.
  • FF em hexadecimal representa o número binário 11111111, que por sua vez representa o número decimal 255.
